Approach

Reading the question, I figured out that keep track of the buying price would be needed to solve the problem, aswell of the best sell price, but, mistakenly, thought it would be necessairy to memorize the sell price, which cost me some development time.

Complexity

  • Time complexity:
    O(n)O(n)O(n);

  • Space complexity:
    O(n)O(n)O(n)

Code

class Solution {
public:
    int maxProfit(vector<int>& prices) {
        int buyPrice = prices[0], bestSell = 0 - buyPrice;
        for (const int price : prices) {
            if (buyPrice > price) buyPrice = price;
            if (price - buyPrice >= bestSell) bestSell = price - buyPrice;
        }
        return bestSell >= 0 ? bestSell : 0;
    }
};
